ไม่สามารถโหลดข้อมูลจาก Google Sheet สำเร็จ (Apps Script Error)

เกิดข้อผิดพลาดขึ้นเนื่องจาก Google Apps Script คืนผลลัพธ์เป็นโครงสร้าง HTML หรือระบบปิดกั้นการเข้าถึง ปัญหานี้แก้ไขได้ง่ายๆ โดยการตั้งค่า Deploy ของคุณให้เป็น "ทุกคน"

🛠️ วิธีแก้ไขแบบเร่งด่วน:

  1. เปิดไปที่ Google Sheets หน้าแก้ไขสคริปต์ (Apps Script)
  2. กดปุ่ม การทำให้ใช้งานได้ (Deploy) -> จัดการการทำให้ใช้งานได้ (Manage Deployments)
  3. กดรูปดินสอ และตรวจดูที่ช่อง "ผู้ที่มีสิทธิ์เข้าถึง" (Who has access)
  4. เปลี่ยนค่าจากเดิมให้เป็น "ทุกคน" (Anyone) เสมอ
  5. กด "ทำให้ใช้งานได้" และตรวจเช็ค URL ในระบบให้ถูกต้อง
